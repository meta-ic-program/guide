# Meta IC Engagement Guide

A public reference website for CWAMs (Contingent Workforce Account Managers) at Meta explaining the Independent Contractor (IC) Engagement Process through MBO Partners.

## Deployment to GitHub Pages

1. Create a new GitHub repository (e.g., `meta-ic-guide`)
2. Upload the `index.html` file to the repository
3. Go to **Settings → Pages**
4. Under **Source**, select **Deploy from a branch**
5. Select **main** branch and **/ (root)** folder
6. Click **Save**
7. Your site will be live at `https://YOUR-USERNAME.github.io/meta-ic-guide/`

## Contents

The site covers:
- What is an Independent Contractor (IC)?
- IC vs. W2 Contractor comparison
- 3 Engagement Pathways & costs (FQ IC, Business Services, Payroll/W2)
- 9-step onboarding process
- Equipment ordering, setup & return instructions
- Invoicing & payment timelines
- FAQ (8 questions)
- Support & contacts directory
- AI chatbot (IC Program Assistant)
